# Foodship_Seller_App 
 
## Seller Application - Mobile-Based Applications Management Restaurant  (FoodShip)

### Feature

- Login and Authentication 
- Manage the cart
- Manage user(waiter) : block or active account
- Manage table
- History Order
- Total earning
- Manage all food in restaurant
